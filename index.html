Structure

The stucture of the website must always consist of the following:

  1. Navigation
  2. Heading
  3. Body
  4. Footer

Text Styles

The text of the website must always adhure to the below:

  1. H1: Main Headings - The the main headings must be of the h1 html element.
    • Font Weight: 700 (Bold)
  2. Font Size: 26px
  3. Font-family: Arial, Helvetica, sans-serif
  4. H2: Sub-Headings - The headings immediately after the main headings must be of the h2 html element.
    • Font Weight: 500
    • Font Size: 18px
    • Font-family: Arial, Helvetica, sans-serif
  5. P: Paragraph text - All paragraph text must adhere to the bellow:
    • Font Weight: 400 (Regular)
    • Font Size: 14px
    • Font-family: Arial, Helvetica, sans-serif
